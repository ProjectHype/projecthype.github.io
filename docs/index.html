Hype — Language documentation

A short, friendly reference for Hype: syntax, examples, and a small in-browser playground.

Introduction

Hype is a minimal, readable language designed for teaching and prototyping. It blends familiar C-like control flow with Python-like readability.

Getting started

Save your source in .hy files. A simple CLI tool hypec compiles to a hypothetical bytecode or transpiles to JavaScript (TBA).

File example

// hello.hy
print("Hello, Hype!")

Compile

$ hypec build hello.hy -o hello.js

Syntax

Hype aims to be concise. Key features:

  • Significant whitespace optional — use braces for blocks.
  • First-class functions and simple ADTs.

Comments

// single-line comment
/* multi-line
   comment */

Variables & assignment

let x = 10
mut count = 0
const PI = 3.14159

Functions

fn add(a, b) {
  return a + b
}

print(add(2,3))  // 5

Types

Primitive types: int, float, string, bool. Collections: list, map.

let s: string = "hello"
let arr: list[int] = [1,2,3]
let m: map[string,int] = {"a":1}

Control flow

if x > 5 {
  print("big")
} else {
  print("small")
}

for i in 0..10 {
  print(i)
}

while cond {
  // loop
}

Standard library

Common utilities: print(), map, reduce, io.read(). More coming soon.

Examples

Factorial

fn fact(n) {
  if n <= 1 { return 1 }
  return n * fact(n-1)
}
print(fact(6))  // 720

FizzBuzz

for i in 1..100 {
  if i % 15 == 0 { print("FizzBuzz") }
  else if i % 3 == 0 { print("Fizz") }
  else if i % 5 == 0 { print("Buzz") }
  else { print(i) }
}
